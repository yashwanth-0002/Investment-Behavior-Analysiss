# Investment Behavior Analysis

## 📊 Overview
This project analyzes investor behavior using survey data.

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## 📁 Dataset
- 40 respondents
- 24 variables

## 🔍 Key Insights
- Gold is the most preferred investment
- Investors are risk-averse
- Majority expect 20–30% returns
- Financial consultants are primary source
- Average duration ~3 years

## 💡 Business Insights
- Need for financial awareness
- Opportunity for advisory platforms
